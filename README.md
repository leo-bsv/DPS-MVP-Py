# DPS-MVP-Py
User and resource management for DPS-MVP-Py

# Запуск окружения

### создание окружения

``python3 -m venv venv``

### вход в окружение

``source venv/bin/activate``

### скачивание покетов

``pip install -r requirements.txt``

### запуск docker оркружения

``docker compose up --build``
